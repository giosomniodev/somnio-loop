# somnio-loop

**Ticket in. Deliverable out.**

somnio-loop is a Claude Code plugin that runs a full agentic loop end-to-end without further input. Hand it a ticket — a free-text prompt, a Jira/Linear/GitHub Issues ID, or a paragraph pasted directly — and it handles triage, parallel sub-agent fan-out, self-healing verification, spec generation, and ADR recording automatically. It is technology-agnostic and works on any stack with a discoverable manifest.

For the current release, see the [CHANGELOG](https://github.com/giosomniodev/somnio-loop/blob/main/CHANGELOG.md).
